## 📝 CONVENÇÕES DESTE PROJETO

### Nomenclatura:
- **Componentes:** PascalCase (`Header.tsx`)
- **Funções:** camelCase (`fetchData`)
- **Constantes:** UPPER_SNAKE_CASE (`API_URL`)

### Commits:
```

fix: corrige bug
```
